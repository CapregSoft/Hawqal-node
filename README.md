# Hawqal

This package that contains the data of world's countries,states and their cities name

## Functionalities

- Get Countries
- Get States 
- Get Cities

## Get Started
To install the Hawqal-go package, use the following command:
```golang
go get github.com/CapregSoft/Hawqal-node
```
